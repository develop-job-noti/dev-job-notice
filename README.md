## Environment
- Backend: Python 3.12.3, Django 5.0.4, DRF 3.15.1
- Database: SQLite3

## Getting Start
1. .env 파일 확인 (담당자한테 요청)
2. 가상환경 셋팅 `python -m venv .venv`
    1. Python 3.12.3 version
    2. pip 24.0 version
2. 가상환경 적용 `source .venv/bin/activate`
3. 패키지 설치 `poetry install`
4. 서버 실행 `make start`

## Deploy
1. .env 파일 확인 (담당자한테 요청)
2. 배포 스크립트 실행 `make deploy`
